# Prompt Squad 3

AI-powered chat assistants for various tasks and purposes. Interact with specialized AI mascots to get help with writing, analysis, advice, and more.

## Production URL
[https://prompt-squad-3.vercel.app/](https://prompt-squad-3.vercel.app/)

## Features
- **Specialized Mascots**: Expert AI agents for coding, writing, UX design, and more.
- **Multi-Model Support**: Access Gemini, OpenAI, Claude, Perplexity, and Grok.
- **Live Web Search**: Integrated web capabilities for up-to-date information.
- **Deep Thinking**: Enhanced reasoning capabilities for complex problems.

## Tech Stack
- **Frontend**: React Native via Expo (Web & Mobile)
- **Backend**: Supabase (Auth, Database, Edge Functions)
- **AI**: Integration with multiple LLM providers
